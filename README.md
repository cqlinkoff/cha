# Cha 🍵

> A Set of React Components

## Installation

```bash
npm install @cqlinkoff/cha
```

## Use

### example

```js
import { Scroll } from '@cqlinkoff/cha'

export default App extends React.PureComponent {
  render () {
    return (
      <Scroll>
        <div></div>
      </Scroll>
    )
  }
}
```

See [Components](#components) docs for more detail
